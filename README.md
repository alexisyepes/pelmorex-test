## Prerequisites
- Node.js 16+ installed
- Redis server running locally (`redis-server`)

## Backend
At the root of the project run: `npm install`

## Frontend
- "cd" to pelmorex-test-frontend
- Run: `npm install`

## To Run the application
- Make sure that redis is running
- At the root of the project run: `npm run dev` (to start the backend server)
- Open a new terminal, navigate to pelmorex-test-frontend
- Run `npm run dev`
